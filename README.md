# letgo-app

🗑️ **LetGo** は、ミニマリストのための「手放しログ」アプリです。

不要なモノを手放す習慣をサポートし、
・何をいつ手放したか  
・合計でどのくらい手放したか  
をシンプルに記録＆見返せます。

---

## 📱 主な機能

- ✅ 手放しログの登録（品目・カテゴリ・数量・方法）
- ✅ 累計個数の可視化
- ✅ カテゴリ別統計（円グラフ）
- ✅ 31日チャレンジモード
- ✅ ギャラリー表示（写真つき）
